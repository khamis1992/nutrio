# إضافة حفظ مخالفات جميع المركبات PDF

إضافة Chrome محلية تجمع مركبات المنشأة من جميع صفحات القائمة، ثم تحفظ مخالفات كل مركبة في ملف PDF مستقل بمقاس A4 القياسي `210 × 297 مم`.

## تحديث الإضافة

إذا كانت النسخة التجريبية مثبتة بالفعل:

1. افتح `chrome://extensions`.
2. ابحث عن **حفظ مخالفات جميع المركبات PDF**.
3. اضغط زر **إعادة التحميل**.
4. تأكد أن رقم الإصدار الظاهر هو `0.2.1`.

للتثبيت لأول مرة، فعّل وضع المطور واضغط **تحميل إضافة غير مضغوطة**، ثم اختر:

`C:\Users\khamis\Documents\nutrio\tools\moi-traffic-pdf-extension`

## التشغيل

1. افتح صفحة قائمة مركبات المنشأة في موقع وزارة الداخلية. يجب أن ينتهي الرابط بـ:
   `/moipay/inquiry/violation/companyPlateCount`
2. اضغط أيقونة الإضافة.
3. اضغط **اختيار المجلد وبدء حفظ الجميع**.
4. اختر المجلد:
   `C:\Users\khamis\Desktop\المخالفات المرورية`
5. اترك تبويب الوزارة ولوحة الإضافة مفتوحين حتى تظهر رسالة الاكتمال.

## طريقة العمل

- تجمع الإضافة المركبات من جميع صفحات النتائج قبل بدء الحفظ.
- تفتح مركبة واحدة في كل مرة وتنتظر اكتمال بيانات المخالفات.
- تنشئ PDF عموديًا بحجم A4 ومقياس محتوى `100%` وخلفيات الصفحة مفعّلة.
- تحذف أول `00` من رقم المركبة؛ مثل `002766` ← `2766.pdf`.
- تتخطى أي ملف موجود مسبقًا بالاسم نفسه ولا تستبدله.
- تعرض أعداد الملفات المحفوظة والمتخطاة والمتعذرة.
- يمكن إيقاف العملية؛ عند التشغيل مرة أخرى ستتخطى الملفات المحفوظة وتكمل المتبقي.

## ملاحظات مهمة

- لا تغلق لوحة الإضافة أو تبويب الوزارة أثناء التشغيل.
- لا تتنقل يدويًا داخل تبويب الوزارة حتى انتهاء العملية.
- سيعرض Chrome تنبيهًا بأن الإضافة بدأت تصحيح أخطاء التبويب؛ هذا مطلوب لإنشاء ملفات PDF مباشرة.
- إذا تعذر حفظ مركبة واحدة، تسجل الإضافة الخطأ وتحاول متابعة بقية المركبات.
- يتحقق الإصدار `0.2.1` من ظهور جدول المركبات بعد الرجوع، ولا يعتمد على حالة تحميل التبويب وحدها.

## الخصوصية

لا ترسل الإضافة بيانات المركبات إلى أي خادم. إنشاء ملفات PDF والكتابة إلى المجلد يحدثان محليًا داخل Chrome.
