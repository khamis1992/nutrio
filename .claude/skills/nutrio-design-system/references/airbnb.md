# Airbnb Design Patterns

## Single Accent
- One brand color (#ff385c "Rausch") carries every CTA, search orb, heart save, brand links
- 90% of surfaces are white + ink with one or two Rausch moments
- No secondary brand color in mainline marketing

## Pill Shapes
- Buttons: 8px radius
- Search bar: fully pill-shaped (9999px)
- Interactive elements: all rounded, no hard corners

## Photography-Driven
- Trusts photography for visual weight, not typographic muscle
- Property cards: photo-first, aspect-ratio rectangles with rounded corners

## Shadows
- Single shadow tier: `rgba(0,0,0,0.02) 0 0 0 1px, rgba(0,0,0,0.04) 0 2px 6px, rgba(0,0,0,0.1) 0 4px 8px`
- No progressive elevation tiers — one shadow or none

## Whitespace
- 8px base spacing
- Major sections: 64px
- Generous but not editorial-magazine — marketplace density wants more cards per scroll

## Typography
- Display weights modest: 500-700, not heavy 700+
- Rating display at 64px is the only "loud" typographic moment
