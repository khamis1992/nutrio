# Daily Operations Checklist

## Morning Routine (9:00 AM - 10:00 AM)

### System Health Check
- [ ] Check platform uptime (should be 100%)
- [ ] Review error logs from last 24 hours
- [ ] Check Supabase dashboard for any issues
- [ ] Verify payment gateway (Sadad) is operational
- [ ] Test WhatsApp notification delivery

### Order Review
- [ ] Review overnight orders (10 PM - 9 AM)

## Emergency Protocols

### System Down
1. [ ] Check Supabase status page
2. [ ] Check hosting status
3. [ ] Post status update on social media
4. [ ] Notify restaurants via WhatsApp
5. [ ] Work on fix immediately
6. [ ] Send apology + credit to affected customers

### Payment Issues
1. [ ] Check Sadad status
2. [ ] Contact Sadad support if needed
3. [ ] Enable manual payment processing
4. [ ] Notify affected customers
5. [ ] Document all failed transactions

### Restaurant Emergency
1. [ ] Restaurant can't fulfill orders
2. [ ] Contact restaurant immediately
3. [ ] Offer customers alternatives
4. [ ] Issue refunds if needed
5. [ ] Follow up with restaurant

### Customer Complaint Escalation
1. [ ] Acknowledge within 15 minutes
2. [ ] Gather all information
3. [ ] Offer immediate resolution
4. [ ] If complex: escalate to manager
5. [ ] Follow up within 24 hours
6. [ ] Document lesson learned
