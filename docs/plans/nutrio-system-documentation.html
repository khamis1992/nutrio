Security

Authentication & Access Control

Supabase Auth with IP geo-restriction and role-based route protection.

🔐
Auth Flow
// User signs up/in via Supabase Auth await signUp(email, password, fullName) await signIn(email, password) → IP check → session // Session stored in Capacitor Preferences (native) // or localStorage (web)
🌍
IP Geo-Restriction
// Before login, check IP location await checkIPLocation() → allowed: Qatar only → blocked IPs logged in blocked_ips → failed checks logged in user_ip_logs
🛡️
ProtectedRoute
// Role hierarchy: customer(1) < partner(2) < admin(4) ProtectedRoute({ requiredRole: 'partner', requireApproval: true // for restaurants }) → checks user_roles table → derives partner/driver from relationship tables → 5-minute cache for role lookups
