## Brand Guidelines

### Tone
- **Primary:** Encouraging, empowering, supportive
- **Secondary:** Professional, trustworthy
- **Avoid:** Judgmental, overly technical, salesy

### Visual Style
- Clean, modern aesthetic
- High-quality food photography
- Qatar local context (landmarks, people, culture)
- Green/natural color palette

### Voice
- Use "you" - speak directly to audience
- Keep it conversational
- Include Arabic phrases where appropriate (optional)
- Emojis for engagement but not overuse

---

## Best Practices

### For Maximum Engagement
1. **Post consistently** - 3-5 times per week
2. **Use Stories daily** - Behind-the-scenes, polls, updates
3. **Engage in comments** - Reply within 2 hours
4. **Use Reels** - Algorithm prioritizes video content
5. **Save best content** - Create highlights by category

### Content Mix Recommendation
| Content Type | Percentage |
|--------------|------------|
| Reels (Lifestyle/Trendy) | 40% |
| Carousels (Educational) | 25% |
| Single Images (Promo) | 15% |
| Stories | Daily |
| User-Generated Content | 20% |
