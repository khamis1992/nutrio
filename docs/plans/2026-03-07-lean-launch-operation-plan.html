29 Step 1: Partnership with Restaurants

Restaurant Onboarding Process

  • Approach health-focused restaurants in Qatar
  • Each restaurant sets their base meal price (e.g., 50 QAR)
  • Agree on fixed discount: 18% commission per meal
  • Restaurant receives 82% of meal price (e.g., 41 QAR)
  • Fast payment: Net-7 days (vs competitors' 30-60 days)
  • 0% commission for first 3 months (incentive for early partners)

30 Step 2: Sell Subscription to Users

User Subscription Flow

  • User chooses a monthly package (Elite, Healthy, Fresh, or Weekly)
  • Full payment upfront at time of purchase
  • Example: 3,800 QAR for 60 meals + 40 snacks
  • Cash collected before service delivered (no collection risk)
  • Subscription activates immediately

  • No refund for unused meals (unless exceptional circumstances)
  • This creates a win-win regardless of usage level

33 Step 5: Delivery Mechanism

Delivery Operations Overview

Nutrio has an existing fleet and drivers already in place — this is a significant competitive advantage that solves the hardest part of food delivery. The delivery process is fully integrated with the app and driver portal.

Automatic Branch & Distance-Based Selection

Smart Delivery Routing: Nutrio uses an intelligent system that automatically selects the nearest restaurant branch for each order to optimize delivery efficiency and costs.

How It Works

  1. Restaurant Branch Data: Each restaurant in the app has all branch locations registered with precise GPS coordinates
  2. Distance Calculation: When customer selects a meal, the app calculates distance between customer location and each restaurant branch
  3. Auto-Selection: System automatically selects the nearest branch for delivery
  4. Multi-Restaurant Orders: If ordering from 2+ restaurants on the same day, system picks nearest branch for each, then combines orders into one driver route with optimal sequence

Benefits

  • Reduced Travel Time: Each driver travels shorter distances
  • Lower Costs: Reduced fuel and delivery costs
  • Cost Cap: Ensures delivery cost doesn't exceed limit (250–300 QAR)
  • Efficiency: Multiple orders from different restaurants consolidated into single delivery route

Delivery Process Diagram

Figure: End-to-End Delivery Process Flow

Delivery Process Flow

  • Order Confirmed: Restaurant receives order notification via partner app
  • Smart Branch Selection: System auto-selects nearest restaurant branch based on customer location
  • Preparation (15-25 min): Restaurant prepares the meal
  • Ready for Pickup: Restaurant marks order ready, system dispatches driver
  • Driver Assigned: Nearest available driver receives pickup request
  • Route Optimization: If multiple orders, system creates optimal delivery route
  • Pickup: Driver arrives at restaurant, confirms pickup in app
  • In Transit: Real-time tracking begins, user sees driver location
  • Delivered: Driver arrives, confirms delivery (with photo proof optional)
  • Order Complete: Customer notified, meal deducted from balance
